# Scrobble

Minimal Story-side scrobble contract workspace for Pirate.

## Scope

- canonical track registration by stable `trackId`
- direct user scrobbles
- delegated operator-assisted scrobbles
- no onchain ranking, anti-fraud, lyrics, covers, or playlist state
